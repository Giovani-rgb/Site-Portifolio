Metodologias Ágeis

Scrum

Scrum é uma metodologia ágil utilizada principalmente para gerenciamento e desenvolvimento de software. Ele organiza o trabalho em sprints, onde são realizadas reuniões diárias (Daily Scrum) para monitorar o progresso.

Kanban

Kanban é uma metodologia ágil que utiliza um sistema visual para gerenciar o trabalho à medida que avança em um processo. O foco está na entrega contínua e na melhoria incremental.

Ferramentas de Qualidade

PDCA

PDCA é um método cíclico de gestão de processos que visa a melhoria contínua de produtos, processos ou serviços.

Matriz SWOT

A Matriz SWOT é uma ferramenta utilizada para realizar a análise de cenários, com base nas forças, fraquezas, oportunidades e ameaças do negócio.
